### mac ntfs读写工具安装
1. 安装homebrew
2. 安装macFuse （安装后重启生效）
下载地址：https://osxfuse.github.io/
页面右侧有下载链接。
3. 安装ntfs-3g
```
brew tap gromgit/homebrew-fuse
brew install ntfs-3g-mac
```
4. 安装mounty（参考官网https://mounty.app/）
```
brew install gromgit/fuse/mounty
```
或者直接下载 mounty的dmg安装包
