1. 安装插件 .ignore，重启生效
2. 项目根目录右击 -> new -> .ignore file -> .gitignore file -> 使用模板
3. 打开.gitignore添加需要忽略的文件

Q:
打开.gitignore如果提示
found ignored but not excluded file  类似的问题，是因为.gitignore只会忽略未被git add的文件，
即未被追踪的文件，而你要忽略的文件已经被add到git 仓库了。
解决：git rm -r --cached .
git add .
git commit -m ""
说明：删除暂存区的文件，然后重新add即可
